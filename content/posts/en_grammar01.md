### 2. 8품사
1. 명사
2. 대명사
3. 동사
4. 형용사
5. 부사, 전치사, 접속사, 감탄사

### 3. 왜 해석이 안되나(→ 구문이 복잡하고 길어서, 구 또는 절)
* 구 : 단어 + 단어 = 품사 / 성분
  * 명사구 → to + 동사원형 / 동명사 / 의문사 + to V
  * 형용사구 → to + 동사원형 / 분사
  *  부사구 → to + 동사원형
* 절 : S + V 문장 = 품사 / 성분
  * 명사절 → if, whether절 / that절 / 관계대명사 what절 / 의문사절
  * 형용사절 → 관계사
  * 부사절 → if(...라면) / when(...할 때)
